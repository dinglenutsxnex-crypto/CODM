# How to use 

1. Enable desired option in menu
2. start the fight 
3. Simply leave the fight

Rn all mods work, duels are kinda janky and refuse to work sometimes.

## Building

Requires JDK 17 and the Android SDK.

```
./gradlew assembleDebug
```

CI (`.github/workflows/build.yml`) builds a release APK on every push and uploads it as a workflow artifact.
