# HammerScale

Vpn based hack for sf3 

![Demo](https://raw.githubusercontent.com/dinglenutsxnex-crypto/THE/refs/heads/main/Readme_assets/ezgif-5cf29e9e8489a9e9.gif)

# How to use 

1. Enable desired option in menu
2. start the fight 
3. Simply leave the fight

## Building

Requires JDK 17 and the Android SDK.

```
./gradlew assembleDebug
```

CI (`.github/workflows/build.yml`) builds a release APK on every push and uploads it as a workflow artifact.
